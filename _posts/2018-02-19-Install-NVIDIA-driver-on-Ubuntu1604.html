--- layout: post title: Ubuntu 16.04 Desktop에서 NVIDIA 최신 driver 쉽게 설치하기 subtitle: Install NVIDIA driver on Ubuntu 16.04 desktop ---

Ubuntu 16.04 Desktop 버전에서 NVIDIA 최신 드라이버를 나름 쉽게 설치하는 방법입니다.

1. 터미널에서 다음과 같이 입력합니다. 

$ sudo add-apt-repository ppa:graphics-drivers/ppa
$ sudo apt-get update

2. "Additional Drivers"를 실행합니다.출처:
Install Nvidia Driver on Ubuntu 16.04 (GUI Method for Beginners) 2번 항목부터는 출처에 있는 내용과 거의 같습니다. 최신 버전이 나오는 것만 다릅니다.

3. "Using NVIDIA binary driver - version xxx.xx from nvidia-xxx" 을 선택합니다. 이 과정은 몇 분 걸립니다. 여기에서 xxx.xx 나 xxx 는 버전을 가리키는 숫자입니다. 참고로 저는 CUDA 9.1을 설치하기 위해출처: CUDA Toolkit 9.1 Download 390.xx 버전을 설치했습니다.

4. 드라이버 설치가 끝나면 재부팅합니다.
